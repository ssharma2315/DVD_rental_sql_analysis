# DVD Rental SQL Analysis

This project analyzes the PostgreSQL DVD Rental dataset to answer business-oriented questions using SQL.

## Questions Solved

1. Which film categories generate the most revenue?
2. Who are the top 10 customers by lifetime spend?
3. Which months had the highest and lowest rental activity?

## Tools Used

- PostgreSQL
- SQL
- Git
- GitHub

## Project Structure

queries/ → SQL queries  
results/ → Query outputs
